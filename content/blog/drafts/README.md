# Drafts

Esta pasta será usada para armazenar rascunhos gerados automaticamente pelo pipeline editorial.

Fluxo esperado:

1. o worker diário coleta sinais e notícias relevantes
2. um agente gera possíveis posts em Markdown
3. os arquivos de draft são adicionados nesta pasta via Pull Request
4. a mesma PR tambem atualiza `content/blog/published` com a versao pronta para publicar

Nenhum arquivo nesta pasta deve ser tratado como post público por padrão.
